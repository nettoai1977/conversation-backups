# Sample Asset Design Templates

## Design Template 1: "Mountain Silhouette"
```
    /\      /\      /\
   /  \    /  \    /  \
  /    \  /    \  /    \
 /      \/      \/      \
---------------------------
```

Description: Clean black mountain silhouette against white background. Single line art style with three peaks of varying heights. Works well in both horizontal and vertical orientations.

Color Variations:
- Black on white (classic)
- Sage green on cream
- Charcoal on light gray

Applications: T-shirts, hoodies, posters, stickers, tote bags

## Design Template 5: "Motivational Minimalist"
```
BREATHE
```

Typography: Clean, sans-serif font with generous spacing. Letter height approximately 3-4 times the width of a character. Center-aligned with significant white space around.

Color Variations:
- Black on white
- Sage green on cream
- Charcoal on light gray

Applications: Yoga apparel, wellness products, meditation accessories
